03 · How I Started in IT

It began with a simple question: how are websites and apps actually made?

That first curiosity turned into the path I am on now. Each step added a little more depth, from structure, to interaction, to recreating premium interfaces with intention.

From curiosity to practice

My entry into programming was driven by fascination.

I was fascinated by how websites and applications are created. I started learning HTML and CSS, then moved on to JavaScript and modern frontend development.

The more projects I built, the more interested I became in creating interactive user interfaces and recreating premium designs from companies like Porsche.

Step 01

Curiosity about digital products

I became interested in the process behind websites and applications, which made programming feel worth exploring seriously.

Step 02

Starting with HTML & CSS

The first stage was learning how to structure pages and style them properly through frontend fundamentals.

Step 03

Moving into JavaScript

After the basics, I began working with JavaScript to make websites more interactive and dynamic.

Step 04

Focusing on modern frontend work

With each new project, I became more interested in responsive interfaces, smooth interaction, and polished presentation.

Step 05

Recreating premium design language

Premium brand-inspired interfaces, especially Porsche-style presentation, pushed me to aim for higher visual quality.
