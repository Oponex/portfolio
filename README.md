# Porsche-Inspired Portfolio Website

> A premium personal portfolio inspired by Porsche's design philosophy — clean, elegant, fast, and built with attention to every detail.

---

## About

This project is more than just a portfolio.

It represents my journey as a frontend developer, documenting how I progressed from writing my first lines of HTML to building interactive, premium-quality web experiences inspired by one of the world's most iconic automotive brands.

Instead of creating a traditional portfolio, I challenged myself to recreate the feeling, quality, and attention to detail found on the official Porsche website while keeping the project completely original.

Every animation, transition, layout, and interaction was carefully designed to create a smooth and immersive user experience.

---

## My Journey

My programming journey started with curiosity.

I wanted to understand how modern websites worked and how professional interfaces were built.

At the beginning I only knew basic HTML and CSS.

After learning the fundamentals, I started exploring JavaScript and interactive user interfaces.

With every project I learned something new.

Instead of copying templates, I challenged myself by recreating premium interfaces that required careful thinking about layout, responsiveness, animations, and user experience.

This project represents months of continuous learning, experimentation, and improvement.
